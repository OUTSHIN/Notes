## Functions

### Characteristics

- has a **name**
- has **parameters** (0 or more)
- has a **docstring** (optional but recommended)
- has a **body**
- **returns** something

### Framework

```python
def <name> ( <paraments> ):
    """
    specification docstring
    """
    <expression>
    <expression>
	return <expression>
```

#### lambda函数

> <函数名>=lambda( <参数> ) : <表达式>
